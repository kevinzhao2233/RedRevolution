## 对开发人员

如果你是开发人员，或者需要在本地运行本网站，请看下面

clone:

```sh
git clone https://github.com/kevinzhao2233/RedRevolution.git
```

start：

```sh
cd RedRevolution
npm install
npm run dev
```


##### 注意，开发环境需要

 node.js版本在 8.0 以上
